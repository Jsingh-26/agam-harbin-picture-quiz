A fun, educational picture identification game for Agam (4 years old) and Harbin (10 years old)!

## Features
- **Turn-based gameplay** - Alternates between Agam and Harbin
- **Age-appropriate difficulty** - Simple colorful images for Agam, challenging sketches for Harbin
- **Fully responsive** - Works on mobile, tablet, and desktop
- **Beautiful animations** - Confetti celebrations and smooth transitions
- **No sounds** - Visual-only feedback
- **Scoring system** - Track progress for both players

## Installation

```bash
npm install
```

## Run Locally

```bash
# Start Expo development server
npm start

# Run on Android
npm run android

# Run on iOS
npm run ios

# Run on Web
npm run web
```

## Build APK

```bash
# Install EAS CLI
npm install -g eas-cli

# Login to Expo
eas login

# Build Android APK
eas build --platform android --profile preview
```

## Game Rules
- 10 rounds total (5 turns each)
- Each correct answer gives +10 points
- Wrong answers let you try again (no penalty)
- Final scores shown at the end

- React Native
